Structure Alarm​List 

struct AlarmList

A model that represents a list of alarms.

AlarmList holds arrays for alarm identifiers and time components (seconds, minutes, hours, and AM/PM). It provides computed properties for generating formatted labels, converting hours to a 24-hour format, classifying alarms as morning or night, calculating time differences between alarms, and producing a formatted layout string for display.

real​Hour​List 

var realHourList: [Int]

A computed array of converted hours in 24-hour format.

For each alarm, if the day is "AM" and the hour is 12, it is converted to 24; otherwise, for "PM" the hour is increased by 12 (unless it's 12 already).

morn​List 

var mornList: [String]

A computed array classifying each alarm as "Morning" or "Night".

An alarm is considered "Morning" if its converted hour is between 6 (inclusive) and 18 (exclusive); otherwise, it is considered "Night".

diff​List 

var diffList: [Int]

A computed array of time differences (in seconds) between consecutive alarms.

For the first alarm, the difference is 0. For subsequent alarms, the difference is calculated based on the difference in hours, minutes, and seconds between the current and previous alarm.

layout 

var layout: [String]

A computed array that returns formatted time strings for each alarm.

The time is formatted with appropriate zero padding for hours, minutes, and seconds.
